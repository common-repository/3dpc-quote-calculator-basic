== Frequently Asked Questions ==

= Does the plugin offer instant checkout? =

Yes, both Basic and Premium Versions offer Instant Checkout through Paypal. Premium offers Stripe and add-on capabilities.

= How does your plugin work? =

It calculates the volume of a STL file. Premium Version also calculates Density based on filament types.

= Does it support different currencies? =

Basic Version only support USD, EUR, Yen, British Pound, Chinese Yuan, Australian Dollar.
Premium Version supports all basic currencies + Brazilian Real, Canadian Dollar, Czech Koruna, Danish Krone, Hong Kong Dollar, Hungarian Forint, Israeli New Sheqel, Malaysian Ringgit, Norwegian Krone, New Zealand Dollar, Philippine Peso, Polish Zloty, Russian Ruble, Singapore Dollar, Swedish Krona, Swiss Franc, Taiwan New Dollar, Thai Baht

= Can I add, edit, and delete preset filament material? =

Yes, you can.

= How do I translate the plugin? =

= Why does the basic version charge 8% commission? = 

Basic version is meant as a testbed for 3d print professionals to test the plugin. By any chance, a start up can not afford the annual license fee then they can pay a 8% commission fee for every transaction until they can afford the annual license fee. We put a lot of time and effort in developing this plugin and continue to develop. This is our way to make sure those who want a powerful plugin that calculates 3d prints can do so without making an upfront financial commitment.

= Does the plugin support all types of 3D Printers? = 
The plugin calculates volume and density information of a STL 3D file. Therefore, does not matter what 3d printer you use this plugin can calculate any STL file is uploaded.


== Installation ==

= Minimum Requirements =

* WordPress 4.1 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater
* Some payment gateways require fsockopen support (for IPN access)

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/3DP Quote Calculator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 3DP Quote Calculator->Settings screen to configure the plugin
